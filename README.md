<H1>📚과외 웹사이트</H1>
JSP와 Servlet을 이용해서 만든 과외 웹사이트입니다.
<H2>😊목표</H2>

> 1. JSP와 Servlet을 사용하여 게시판을 구현하는 방법을 익힌다.
> 2. MariaDB를 이용하여 DB의 CRUD기능을 연습한다.
> 3. Ajax를 사용하여 서버와 비동기 통신을 한다.
> 4. UML을 사용해서 프로그램의 동작원리를 논리적으로 설명한다.
